# hackathon_diro_2026_public

Ce repo contient les défis et les informations pour les participants du hackathon du DIRO 2026.

Les questions, les bases de codes ou les données de chaque défis se trouvent dans les dossiers spécifiques à chaque défis dans "defis".
Les informations générales et les installations recommandées se retrouvent dans "informations".
La présentation du hackathon se trouve dans "presentation".
